# 📊 HR Analytics Dashboard (Tableau)

## 📌 Project Overview

This project presents a comprehensive **Human Resources Analytics Dashboard** built using **Tableau**, designed to provide both high-level insights and detailed employee-level analysis. The dashboard enables HR managers and stakeholders to make data-driven decisions by exploring workforce trends, demographics, and compensation patterns.

---

## 🎯 Objectives

* Provide a clear overview of workforce metrics
* Analyze employee demographics and diversity
* Identify salary trends and potential disparities
* Enable detailed exploration of individual employee records

---

## 📁 Dataset

The dataset used in this project contains structured HR data, including:

* Employee personal details (age, gender, education)
* Employment information (department, job title, location)
* Salary and performance metrics
* Employment status (active, terminated, hired dates)

---

## 📊 Dashboard Structure

### 🔹 1. Summary View

#### 🧭 Overview

* Total number of **hired, active, and terminated employees**
* Year-wise trends of **hiring and termination**
* Employee distribution by:

  * Department
  * Job titles
* Comparison between **Headquarters (New York)** and branch offices
* Geographic distribution across **cities and states**

#### 👥 Demographics

* Gender ratio across the organization
* Distribution of employees by:

  * Age groups
  * Education levels
* Count of employees in each category
* Correlation between **education level and performance ratings**

#### 💰 Income Analysis

* Salary comparison across:

  * Education levels
  * Gender groups
* Analysis of how **age impacts salary** within each department

---

### 🔹 2. Employee Records View

* Detailed table containing:

  * Name
  * Department
  * Position
  * Gender
  * Age
  * Education
  * Salary
* Fully interactive filters for all columns to support deep analysis
